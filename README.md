# Quotation Builder AI

Industrial Equipment Quotation Builder with AI Integration

## Features

- **Multi-Option Quotations**: Create and compare Option A, B, C configurations
- **Equipment Matrix**: Comprehensive pricing database for industrial equipment (AFS, CONV, SILO, DRY, ETS)
- **LOT-Based Grouping**: Components organized by functional groups
- **Real-time Pricing**: Automatic calculation with material multipliers, labor, and PM costs
- **AI Integration**: OpenAI API support for description generation and smart suggestions
- **Export Ready**: Generate professional PDF quotations

## Quick Start

### Option 1: GitHub Pages
1. Fork this repository
2. Enable GitHub Pages in Settings
3. Access at `https://yourusername.github.io/quotation-builder/`

### Option 2: Local
1. Download `Quotation-Builder-AI-vr33.html`
2. Open in any modern browser
3. No server required - runs entirely client-side

## AI Setup (Optional)

To enable AI features:

1. Go to **Admin Panel** â†’ **AI Settings** tab
2. Enter your OpenAI API key
3. Select your preferred model (GPT-4o recommended)
4. Click **Save API Key**
5. Use **Test Connection** to verify

### Supported Models
- GPT-4o (Recommended)
- GPT-4o Mini (Faster, cheaper)
- GPT-4 Turbo
- GPT-3.5 Turbo (Budget option)

### Security Note
- API keys are stored only in your browser's localStorage
- Keys are never sent to any server except OpenAI
- Clear your key anytime in AI Settings
